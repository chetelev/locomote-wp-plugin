AI-powered tool that automates content creation, scheduling, and publishing directly inside your website.

== Description ==

Locomote is an AI-powered content automation engine for WordPress.  
It lets you automatically generate, customize, schedule, and publish posts using a simple interface — all built directly into the WordPress dashboard.

Perfect for bloggers, businesses, marketing agencies, and anyone who wants to automate publishing while keeping full control of their content.

### Features

- AI-powered blog post generation
- Automatic content scheduling
- Bulk post creation
- Secure WordPress Application Password integration
- Modern and fast admin interface
- Lightweight and easy to use
- No coding required
- Fully works inside the WordPress admin panel

### Application Password Support

Locomote integrates with WordPress Application Passwords to allow secure and automated publishing.

### Getting Started

1. Install and activate the plugin
2. Open "Locomote" in the WordPress admin menu
3. Connect your site and start generating posts
4. Create, schedule, and automate your publishing workflow

== Installation ==

1. In your WordPress dashboard, go to **Plugins → Add New**
2. Search for **"Locomote – AI Content Automation"**
3. Click **Install Now**
4. Click **Activate**
5. Navigate to **Locomote** in the admin sidebar to begin using the plugin

== Frequently Asked Questions ==

= Does Locomote use external APIs? =
Yes. Locomote connects to the Locomote API to generate AI-powered content.  
No analytics or personal data are sent — only the text required for content generation.

= Does it store my passwords? =
Locomote does not store your WordPress login password.  
The plugin only uses WordPress Application Passwords generated by WordPress core.

= Does it work with any theme or builder? =
Yes. Locomote works fully inside the WordPress admin area and is compatible with all themes and page builders.

== Privacy ==

Locomote connects to the Locomote API for generating AI-powered content.  
No personal data, analytics, or tracking information is transmitted.  
Locomote does not collect or sell user data.
